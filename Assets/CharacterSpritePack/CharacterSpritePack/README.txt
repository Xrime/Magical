============================================
 Pixel Character Sprite Pack by Noa
============================================

Thanks for downloading my pixel art character sprite pack!

This pack includes a full set of character animations ready to use in any 2D game.

-------------
 Animations
-------------

- Idle
- Running
- Start Jump
- Falling
- Hurt (White Flash)
- Hurt (Red Flash)
- Dying

All animations are provided as:
- PNG spritesheets (with transparent background)
- Individual PNG frames per animation
- Preview GIFs for reference

 Technical Details
--------------------------

- Frame Size: 256x128 px (adjust if needed)
- Format: PNG (transparent background)
- Recommended framerate: 12 FPS (can be adjusted)
